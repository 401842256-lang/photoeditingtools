# 朋友圈中心封面长图生成器 MVP

这是一个纯前端 MVP，可以直接通过浏览器打开使用。

## 使用方式

推荐启动本地预览：

```bash
python3 -m http.server 5173
```

然后打开：

```text
http://localhost:5173/index.html
```

## 当前已实现

- 上传 1-100 张内容照片
- 导入时显示处理进度
- 选择输出数量：2 / 3 / 4 / 6 / 9
- 选择或额外上传 feed 封面图
- 在封面切分画布中拖拽、缩放、居中封面图
- 按朋友圈布局切分封面：1x2 / 1x3 / 2x2 / 2x3 / 3x3
- 每个封面切片固定为 1:1 正方形，整体组合比例随发布数量变化
- 自动生成中心封面长图
- 内容照片会先自动进入标准容器：横图 3:2、方图 1:1、竖图 2:3
- 自动分组时会优先上下配平，再参考原图横竖方向选择容器
- 单画布编辑一张长图
- 左侧小窗切换不同长图，并显示对应的正方形封面切片
- 右侧显示整张长图缩略预览
- 长图内容图按原图比例展示，避免强行横竖裁切
- 当上下高度不接近时，画布会提示补白风险，并支持基于标准容器高度推荐替换
- 导出时仅把补白作为兜底机制，确保中心封面块位于整张长图正中间
- 导出前如存在明显补白，会提醒用户返回调整或继续导出
- 朋友圈预览会从即将导出的每张长图中心裁切 1:1 后拼合，检查真实发布效果
- 通过拖拽手柄调整顺序
- 在图片本体上拖拽调整裁切位置，通过滑杆、按钮或鼠标滚轮缩放
- 中间编辑画布支持适应宽度、75%、50%、25% 视图缩放；缩放只改变视图，不改变容器尺寸
- 中间编辑画布缩放改为滑杆和百分比显示
- 选中照片后才显示容器框选择：自动、横图、方图、竖图
- 右侧整张长图预览显示完整导出长图，并用红色虚线框标出朋友圈中心裁切区域
- 右侧长图预览渲染使用版本锁，避免拖拽排序后被旧预览覆盖
- 封面裁切支持鼠标滚轮缩放
- 补白优化入口统一为“推荐替换”和“推荐新增”，并给出气泡引导
- 照片素材库展示全部上传照片，并按未使用 / 已使用分区管理
- 已使用照片允许再次拖入长图重复使用
- 替换、删除、从当前长图移除
- 删除后识别上下不对称，并支持自动补齐
- 未使用素材池补图
- 朋友圈 feed 预览弹窗
- ZIP 打包导出全部长图
- 浏览器本地暂存
- 上传时自动压缩大图，降低本地暂存压力

## MVP 边界

- 暂不做 AI 内容识别和美学排序
- 暂不做账号、云端保存和多项目管理
- 当前本地暂存使用浏览器 LocalStorage，适合 MVP 验证；正式版建议升级为 IndexedDB
- 不同社交平台的预览裁切规则可能不同，当前优先按朋友圈中心裁切逻辑设计
